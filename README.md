# File Manipulation

- **Support for Multiple Formats**: Compatible with various file formats.
- **Lightweight and Fast**: Optimized for high performance.

## Installation

Clone the repository to your local machine:

```bash
git clone https://github.com/filthypotato/File-Manipulation.git
```

Navigate to the project directory:

